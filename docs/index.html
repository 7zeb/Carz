About Carz

Carz was originally created to add role-play to my City server, with the ability to purchase a Car (Minecart) and lock it to your player, upgrade the car's speed and manage its fuel. The plugin is configurable to match the needs of any server.

Players are able to interact with Carz signs instead of commands, to allow for a more immersive experience; for example putting a refuel sign in a Petrol station.

Carz can integrate with any Economy plugin which supports Vault to allow for monetary purchases.
