Part 1: The Basics

About Carz

Carz was a plugin created to add role-play to my City server. You are able to purchase a Car (Minecart) and lock it to your player, upgrade your car's speed and manage your fuel. Everything is configurable to match the needs of any server.

When you become the owner of a Car, no other player will be able to enter / break your car.

The players are able to interact with signs instead of commands, to allow for a more immersive experience; for example putting a fuel sign in a Petrol station.

Carz can integrate with any Economy / Vault compatible plugin to allow for monetary purchases.

Carz Commands

Every available command can be found in-game using /carz cmds. The list will be built based on permissions and configuration.

For console commands, use /carzc cmds

/carz fuel

Display's the current cars fuel level. If Fuel is enabled.

/carz details New in v7.0.

Display's the current car's details.

/carz refuel

Make a purchase to fill the fuel in your current car.

No extra permission is required, but if Economy is used, a player must have sufficient funds before successfully purchasing.

/carz purchase

Add an owned car to your inventory. You will have full ownership of the car and no other player will be able to unlock it once placed.

Permission Carz.Purchase is required. If Economy is used, a player must have sufficient funds before successfully purchasing.

/carz upgrade

Upgrade the speed of the Car you are currently sat in. This will play a sound and the car will immediately be increased in speed.

Permission Carz.Upgrade is required. If Economy is used, a player must have sufficient funds before successfully purchasing.

/carz stash

Stash the car you are currently in, back into your inventory. This can also be achieved by hitting the car.

/carz addCB

Add a Material to the list of Climbable Blocks.

Permission Carz.Admin is required.

/carz createtype New in v7.0.

Create a new type of Car. A conversation will be started to allow you to fully customise the car.

Permission Carz.Admin is required.

/carz cartypes New in v7.0.

List the currently available Car types to choose.

Permission Carz.Purchase is required.

/carz economy New in v7.0.

Display all economy related information, including each cost available.

Permission Carz.Admin is required.

/carz spawn

An admin only command which allows a Carz car to be spawned.

Permission Carz.Admin is required.

/carz reload

Reload the Carz configuration file.

Permission Carz.Admin is required.
