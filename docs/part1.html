Part 1: The Basics

Permissions

If you want to use permissions for the plugin, ensure Other.UsePermissions in the config.yml is set to true. Otherwise the players will not require permissions for most functionality.

  • Carz.*
    • All Carz permissions.
  • Carz.Admin
    • Perform Administration commands.
  • Carz.Purchase
    • Ability to purchase your own Car.
  • Carz.Place
    • Ability to place a Car.
  • Carz.Upgrade
    • Ability to upgrade your Car.
  • Carz.CreateSign
    • Ability to create Carz signs.
  • Carz.BypassOwner
    • Bypass the ownership checks of the Car.

Commands

Every available command can be found in-game using /carz cmds. The list will be built based on permissions and configuration.

For console commands, use /carzc cmds

/carz fuel

Display's the current Car's fuel level. If Fuel is enabled.

If BountifulAPI is enabled, this will display as an Action Bar message.

/carz details

Display's the current car's details.

If you have a Car in your hand it will display the Car's metadata.

/carz refuel

Request a purchase to fill the fuel in the current car.

No extra permission is required. If Economy is used a player must have sufficient funds before successfully purchasing. If configured, the cost can scale depending on the amount of fuel left in the car.

/carz purchase [cartype]

Purchase an owned car to your inventory. You will have full ownership of the car, being able to protect and lock it from other players.

You can specify which type of Car you would like to purchase using the parameter, if not specified the "default" car will be used.

Permission Carz.Purchase is required. If Economy is used, a player must have sufficient funds before successfully purchasing.

/carz upgrade

Upgrade the speed of the Car you are currently sat in. This will play a sound and the car will immediately be increased in speed.

Permission Carz.Upgrade is required. If Economy is used, a player must have sufficient funds before successfully purchasing.

/carz stash

Stash the car you are currently in, back into your inventory. This can also be achieved by hitting the car.

/carz createtype

Create a new type of Car. A conversation will be started to allow you to fully customise the car.

Permission Carz.Admin is required.

/carz cartypes

List the currently available Car types to choose.

Permission Carz.Purchase is required.

/carz economy

Display all economy related information, including each cost available.

Permission Carz.Admin is required.

/carz spawn

An admin only command which allows a Carz car to be spawned.

Permission Carz.Admin is required.

/carz reload

Reload the Carz configuration file.

Permission Carz.Admin is required.
