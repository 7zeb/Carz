Part 4: Economy

Description

Carz can link to any Economy plugin that uses Vault. When linked, each Carz purchase type can have an associated cost. Any economy based purchases will be prevented until the player has earned enough money (for example buying a car).

Purchase Confirmation

The player can be asked to confirm purchases before the cost amount is deducted from their account. This acts as a good way to provide a summary of what the user is about to do, for example purchase a car.
